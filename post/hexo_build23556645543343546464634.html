代码逐行解析

python

  • class: Python关键字，用于定义类
  • Greater: 类名，表示”问候器”

python

  • def: 定义方法
  • __init__: 构造方法，创建对象时自动调用
  • self: 指向当前对象实例
  • greeting: 参数，接收问候语

python

  • self.greeting: 在对象上创建属性
  • = greeting: 将参数值赋给属性
  • 作用: 把问候语保存到对象中，让其他方法可以使用

python

  • __call__: 特殊方法，让对象可以像函数一样被调用
  • self: 当前对象实例
  • name: 参数，接收要问候的人名

python

  • f"...": f-string格式化字符串
  • {self.greeting}: 使用对象中保存的问候语
  • {name}: 使用传入的人名参数
  • return: 返回组合后的问候字符串

完整执行流程

python

执行过程：

  1. Python创建 Greater 类的新实例
  2. 自动调用 __init__("Hello")
  3. self.greeting = "Hello" - 把”Hello”保存到对象中
  4. 返回创建好的对象，赋值给 greeter

此时对象状态：

python

python

执行过程：

  1. greeter("Alice") - Python自动转换为 greeter.__call__("Alice")
  2. 执行 __call__ 方法：
    • self.greeting 是 “Hello”（从对象中读取）
    • name 是 “Alice”（从参数传入）
    • 返回 "Hello, Alice!"
  3. print() 输出结果
